# Cadastro Alunos

## Comandos usados
```
sudo npm install -g @vue/cli
vue create web
sudo npm install -g axios

composer global require laravel/installer
echo 'export PATH=$PATH:/$HOME/.config/composer/vendor/bin' >> ~/.bashrc
laravel new api
```

## Startando o projeto
```
execute no terminal:
chmod +x ./install (apenas na primeira vez)
./bin/start
```
